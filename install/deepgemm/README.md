# DeepGemm

DeepGEMM is a high-performance GEMM (General Matrix Multiplication) library optimized for deep learning workloads, as described in the [DeepSeek-V3 Technical Report](http://arxiv.org/abs/2412.19437). It provides efficient FP8/BF16 matrix multiplication operations that are essential for transformer model inference and training.

## Usage

```bash
# Set environment variable if not already set
export CUDA_HOME=/usr/local/cuda
# Install DeepGemm
./install_deepgemm.sh
```
